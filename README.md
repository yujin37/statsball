# statsball
우아한테크코스 8기 프리코스 4-5주차 오픈미션
## 프로젝트 소개
가장 관심있는 야구 데이터를 바탕으로 React와 
Spring을 통해 간단한 정보 사이트를 구현하고자 합니다.
## 사용할 기술
### Frontend
- React
### Backend
- Spring Framework
### db 구축
## 주요 기능 목록
### 선수 기록(투수, 타자)
  - 선수 목록 기본 출력(이름순)
  - 중복 이름일 경우 구단 이름순
  - 이름 검색 기능
### 팀 기록
  - 팀 순위 목록
  - 경기 결과 목록형으로 보기
### 선수 목록
  - 투수/내야수/외야수/포수 선택기능
  - 선택한 포지션 내 선수 목록
## 추가 구현 가능 시 구현 기능
### 선수 기록
- 팀, 포지션 선택 필터
- 타율 순 정렬 필터
### 팀 기록
- 경기 결과를 월별 표시 기능
### 선수 목록
- 선수 선택 시 상세 정보 기능
### 공통
- 타 구단 선수들 추가
